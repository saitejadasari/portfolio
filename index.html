Question Answering Application

I developed a question-answering system using Retrieval Augmented Generation (RAG) with BERT and Pinecone. The project involved indexing text data into Pinecone's vector database, leveraging BERT for semantic vector representations. Utilizing Hugging Face's fine-tuned BERT model on SQuAD, the system performs bi-directional understanding of natural language and extracts answers from provided text. At inference, user questions are tokenized and the retriever finds relevant passages in indexed data using Pinecone. The generator then produces answers in a user-friendly HTML and JavaScript UI, seamlessly integrated with Flask for deployment as a service.

News Category Classification

This project categorizes news articles using a Kaggle dataset of 200,000+ labeled articles across 42 categories. Focused on the top 15 categories, a baseline LSTM model achieves 50% accuracy. Upgrading to a Bidirectional RNN model raises accuracy to 70%. Leveraging a pre-trained BERT model, the final approach achieves over 90% accuracy with minimal adjustments, showcasing the power of transfer learning in text classification.
